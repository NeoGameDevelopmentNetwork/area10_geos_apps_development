`2What's new in GDOS64?`

Date: 29-OCT-2022

	`301Content:`
	>> Bugfixes

`361* Bugfixes`

A 30-year old bug was fixed in Pacman screen saver. The last pixel column was not ereased when Pacman switched the direction.

The spooler menu may currently only be $1508 byte in size, since data for the printer spooler and printer driver are stored in MegaPatch within the extendended GEOS-DACC starting at $d180.
This issue was already fixed in GDOS but for MP3 and MegaAssembler the symbol definition files "SymbTab_MMap" and "ExtSym.MP3" have to be corrected.
